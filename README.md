# Repositorio institucional de DataSet

Este repositorio institucional constituye un nodo centralizado de metadatos diseñado para la recuperación estratégica de activos digitales. Facilita la localización de conjuntos de datos mediante una curaduría sistemática de enlaces hacia repositorios externos y datasets específicos, optimizando la interoperabilidad y el acceso a evidencia empírica para investigaciones avanzadas en Ciencia de Datos e IA.
